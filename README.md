# BuilderX AI Homepage

This repo contains the BuilderX AI homepage. The project was initiated with DIAD and then used to generate documentation and deploy via Vercel, including Vercel builds.

## What this is

- A Next.js-based marketing/homepage for BuilderX AI.
- Maintained as a simple, static-first site with modern UI components.

## Local development

```
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Deployment

- Deploy to Vercel (Github Actions).

## Notes

- The site content lives in `src/app`.
- Static assets live in `public`.
